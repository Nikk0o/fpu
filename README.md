# Floating Point Unit
Synthesizable IEEE 754 floating point unit made in verilog.
Avalible operations:
* conversions between single and double precision
* conversions between integer and floating point (wip)
* adder / subtractor (soon)
* multiplier (soon)
* divider (soon)
* square root (soon ?)
* log (soon ?)

## Testing
