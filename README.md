# Floating Point Unit
Synthesizable IEEE 754 floating point unit.
Avalible operations:
* conversions between single and double precision
* conversions between integer and floating point
* equals and greather than
* adder / subtractor (soon)
* multiplier (wip)
* divider (soon)
* square root (soon ?)
* log (soon ?)
